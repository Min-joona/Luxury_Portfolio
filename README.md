## 🚀 What I Built

This is my personal portfolio website showcasing my work as a Full Stack Developer. I built it from scratch using modern web technologies to demonstrate both my technical skills and attention to detail.

### Key Features

- **Interactive 3D Elements** - Custom chocolate molecule visualization using Three.js that you can interact with
- **Smooth Animations** - Every transition and interaction is carefully crafted with Framer Motion
- **Dark/Light Theme** - Elegant theme switching that adapts the entire UI
- **Blog System** - Full-featured blog where I share my thoughts on development, design, and technology
- **Real-time Engagement** - Blog posts support likes, shares, and comments (stored in MongoDB)
- **Responsive Design** - Perfectly optimized for desktop, tablet, and mobile
- **SEO Ready** - Complete with meta tags, structured data, and sitemap for search engines

## 🛠 Tech Stack

**Frontend:**
- React 19 with Hooks
- Three.js + React Three Fiber for 3D graphics
- Framer Motion for animations
- Tailwind CSS for styling
- React Router for navigation

**Backend:**
- Node.js + Express
- MongoDB with Mongoose
- RESTful API design

**Tools:**
- Git & GitHub
- Vercel for deployment
- MongoDB Atlas for database

## 🏃‍♂️ Running Locally

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/portfolio.git
cd portfolio

# Setup frontend
cd luxury-portfolio/frontend
npm install
npm start

# Setup backend (in another terminal)
cd ../backend
npm install
# Create .env file with your MongoDB URI
npm start
```

Visit `http://localhost:3000` to see it running!
